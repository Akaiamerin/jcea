题目

介绍一下数组的两种初始化格式。

答案

```Java
int[] arr1 = new int[]{1, 2, 3}; //静态初始化
int[] arr2 = new int[3]; //动态初始化
```

题目

介绍一下二维数组的元素访问格式。

答案

```Java
int[][] arr=new int[3][4];
arr[0][0] = 1; //将第一个一维数组的第一的元素赋值为 1
```
